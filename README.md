<h1 style="text-align: center">AUTUMN 管理系统-后端</h1>

#### 项目简介
一个基于 JDK1.8、MySQL8.0、Spring Boot、Mybatis Plus、JWT、Spring Security、Redis、rabbit MQ、Vue2.6、vuex、element-ui、sockJS的前后端分离的管理系统后端源码。
